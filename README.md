# # LyricaLytics: A Data Analytics Project Using Spotify's Database

## Overview
LyricaLytics is a data analysis project focused on understanding music preferences and trends using Spotify streaming data. The project explores hypotheses related to song duration vs popularity, explicit content vs popularity over time, and genre popularity by country.
